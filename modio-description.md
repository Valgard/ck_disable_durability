# Disable Durability

Tools, weapons, and armor in Core Keeper slowly wear down as you use them.
This mod stops that — while it is active, items no longer lose durability.

Each item's durability bar simply stays where it is: an item at 100/100
stays full, an item at 30/100 stays at 30/100. Nothing you carry will break
from use again.

**Not a repair mod.** Disable Durability *freezes* durability — it does not
refill it. Items keep whatever durability they currently have; they will not
regenerate back to maximum. Get an item to full durability once and it stays
full for good.

**Multiplayer:** install the mod on both the client and the server.
